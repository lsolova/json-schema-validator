# JSON schema validator worker

This project is to test and improve JSON schema validation.

## Development

1. Change to _src/wasm_ directory `cd src/wasm`
2. Run `cargo install`
3. Run `./build.sh`
4. Change back to root directory `cd ../..`
5. Run `npm i`
6. Run `npm run build:worker`

### Local testing

To start the application on localhost, run `npm run serve`.
